Visual designer

Als ik ga ontwerpen dan staat de klok even helemaal stil. Ik heb er niet veel meer voor nodig behalve een kop thee en goede muziek! Dan zit ik in mijn eigen ontwerp bubbel. Ontwerpen doe ik nu net een jaar. Ik schets mijn concepten nog ouderwets op papier. Daarna maak ik ze digitaal, dat is ook iets wat ik pas dit jaar heb geleerd. Het omgaan met Adobe programma's en Sketch. Het begon ooit met een robotlogo en nu alles wat je op deze website tegenkomt heb ik ontworpen.

Beginnend codeklopper

Toen ik voor het eerst een website codeerde in HTML en CSS, voelde ik mij een echte webwizard. Wow magisch! Coderen is een belangrijke vaardigheid, vooral voor een visual designer. Ontwerpen die ik bedenk moeten goed te maken zijn voor developers. Zolang ik blijf oefenen met HTML, CSS en JavaScript zal ik ook steeds een beter beeld krijgen van de mogelijkheden. Ik ben nog een echte beginner, maar ik probeer altijd te leren.
